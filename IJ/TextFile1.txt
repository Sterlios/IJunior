1) Heal - метод используется только в одном наследнике, всем он не нужен, а значит в родителе он лишний. 
2) if (_attackCounter % 3 == 0) 
3 - магическое число.
3) public abstract int Attack(); 
боец должен кого-то атаковать, передавайте ему того, чого он атакует. а не возвращайте урон.
4) public int DodgeChance { get; private set; } - свойство нигде снаружи не используется, сделайте приватным полем. так же и остальные свойства наследников.
5) private static Random random = new Random();
Статические моля именуются так s_random
6) проверка на Кто победил?
Ничья в такой проверке недостижима. если у обоих здоровье будет <= 0, то всегда будет выполняться первый if
